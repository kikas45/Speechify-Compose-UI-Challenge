# Speechify Compose UI Challenge

This repository contains a **completed Jetpack Compose UI challenge** for a Books app.  
It demonstrates the implementation of a modern Android app using **Jetpack Compose**, following the challenge instructions and guidelines.

## Features

- MVVM architecture with **ViewModel** and **StateFlow**
- Dependency Injection with **Hilt**
- Navigation with **Compose Navigation**
- Search functionality with live filtering
- Loading states for both list and detail screens
- Book Details screen with **scrollable content**
- Shared element animation for book images
- Proper handling of system insets and padding
- Custom typography usage
